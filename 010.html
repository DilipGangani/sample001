Learn HTML/CSS in simple way.

Two Headings, 1st is normal & 2nd is with CSS style

HEADING with only HTML
HEADING with CSS

				

					--HTML--
<h1>HEADING with only HTML</h1>
<h2>HEADING with CSS</h2>		
				


				

					--CSS--
h2{
border:3px solid #FF0000;
border-radius:10px;
color:#0000FF;
background-color:#FFFF88;
padding-left:10px;
width:290px;
font-family:Arial;
font-size:30px;
}		
				

				
Here CSS style given to <h2> tag so the same style will work on every <h2> tag in the
HTML document.
