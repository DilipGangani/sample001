Learn HTML/CSS in simple way.
We will start with CSS now.
CSS stands for Cascading Style Sheets. CSS can help to make good look for HTML pages
& it can be use by internal & external both way.
To use it internal you have to use following code/script.

					--CSS internal in HTML--
<head>
<style type="text/css">
xx
</style>
</head>		
				

			
and to use it external you have to make new CSS file like "style.css" and save in same
along with HTML file by adding following code/script in HTML file.

				--HTML--
<head>
<link href="style.css" rel="stylesheet" type="text/css" />
</head>
